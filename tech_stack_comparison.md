## üíª Getting Started with Next.js

### Step 1: Install Prerequisites
```bash
# Install Node.js (LTS version)
# Download from: https://nodejs.org

# Verify installation
node --version  # Should show v18+ or v20+
npm --version   # Should show v9+ or v10+

# Install pnpm (faster than npm)
npm install -g pnpm
```

### Step 2: Create Project
```bash
# Create Next.js project
npx create-next-app@latest my-portfolio

# Choose these options:
‚úÖ TypeScript? Yes
‚úÖ ESLint? Yes
‚úÖ Tailwind CSS? Yes
‚úÖ `src/` directory? No
‚úÖ App Router? Yes
‚úÖ Customize import alias? No

# Navigate to project
cd my-portfolio

# Install additional dependencies
pnpm add @next/mdx @mdx-js/loader @mdx-js/react
pnpm add gray-matter reading-time
pnpm add framer-motion lucide-react
pnpm add next-themes  # For dark mode
```

### Step 3: Start Development Server
```bash
pnpm dev

# Open browser to: http://localhost:3000
```

Copy the "PROMPT FOR AI ASSISTANT" section from the main document and paste it into your AI coding assistant to generate all the code.
